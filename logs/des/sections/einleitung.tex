\section{Einleitung}

Dieses Dokument beschreibt die Implementierung eines Demonstrators für das QUIC-Transportprotokoll auf Basis des Java-Frameworks Kwik. Ziel des Demonstrators ist es, ein Verständnis für die API von Kwik zu erlangen und zentrale Konzepte von QUIC anhand einer konkreten Anwendung nachzuvolziehen.
